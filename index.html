Info about csgencraft

csgencraft is a fast 1.16.5 Spigot server that allows up to multiple players at a time.
The server is currently in a private beta but we may release this as an open-sourced project in the future.

Compatible

csgencraft currently supports: Minecraft Java Edition, Xbox, Switch, PS4, iOS, Android. This is all thanks to GeyserMC and Floodgate!

Stable

Using proper techniques, we have allowed a self hosting method in order to get the fastest speeds as possible. Utilizing 4GB of ram, the server will be able to withstand massive troubles.

Secure

Using a custom server IP, we have made it so none of your IP addresses will show. This method we are using is extremely secure in order to protect your data.
